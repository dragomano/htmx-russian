Суть htmx заключается в наборе атрибутов, которые позволяют вам отправлять AJAX-запросы непосредственно из HTML:

|               Атрибут               |                                      Описание |
| :---------------------------------: | --------------------------------------------: |
|    [hx-get](/attributes/hx-get/)    |    Отправляет `GET`-запрос по указанному URL. |
|   [hx-post](/attributes/hx-post/)   |   Отправляет `POST`-запрос по указанному URL. |
|    [hx-put](/attributes/hx-put/)    |    Отправляет `PUT`-запрос по указанному URL. |
|  [hx-patch](/attributes/hx-patch/)  |  Отправляет `PATCH`-запрос по указанному URL. |
| [hx-delete](/attributes/hx-delete/) | Отправляет `DELETE`-запрос по указанному URL. |

Каждый из этих атрибутов принимает URL для отправки AJAX-запроса. Элемент отправит запрос указанного типа по заданному URL, когда он будет [активирован](/ajax/triggers/):

```html
<button hx-put="/messages">
  Поместить в сообщения
</button>
```

Это говорит браузеру:

> Когда пользователь нажимает на эту кнопку, отправь PUT-запрос по URL `/messages` и загрузи ответ в кнопку.
