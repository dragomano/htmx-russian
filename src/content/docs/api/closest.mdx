Находит ближайший подходящий элемент в родительской иерархии заданного элемента, включая сам элемент.

## Параметры

* `elt` — элемент, с которого начинается поиск селектора
* `selector` — селектор для поиска

## Пример

```js
// находим ближайший оборачивающий div элемента с id 'demo'
htmx.closest(htmx.find('#demo'), 'div');
```
