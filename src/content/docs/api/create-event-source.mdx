Свойство, используемое для создания новых источников событий, отправляемых сервером ([Server Sent Events](https://github.com/bigskysoftware/htmx-extensions/blob/main/src/sse/README.md)). Его можно обновить для предоставления пользовательской настройки SSE.

## Значение

`func(url)` — функция, которая принимает строку URL и возвращает новый `EventSource`

## Пример

```js
// переопределяем источники событий SSE, чтобы не использовать учётные данные
htmx.createEventSource = function(url) {
  return new EventSource(url, {withCredentials:false});
};
```
