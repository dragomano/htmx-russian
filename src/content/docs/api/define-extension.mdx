Определяет новое [расширение](https://htmx.org/extensions) htmx.

## Параметры

* `name` — имя расширения
* `ext` — определение расширения

## Пример

```js
// определяем простое расширение, которое записывает имена всех вызванных событий
htmx.defineExtension("silly", {
  onEvent : function(name, evt) {
    console.log("Событие " + name + " сработало!")
  }
});
```
