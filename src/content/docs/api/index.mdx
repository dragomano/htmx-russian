Хотя это не является основной целью библиотеки, htmx предоставляет небольшой API вспомогательных методов, предназначенных в основном для [разработки расширений](https://htmx.org/extensions) или работы с [событиями](/events).

Проект [hyperscript](https://hyperscript.org/) предназначен для предоставления более обширной поддержки скриптов для приложений на основе htmx.
