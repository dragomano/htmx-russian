Логгер, который htmx использует для записи информации.

## Значение

`func(elt, eventName, detail)` — функция, которая принимает элемент, имя события и детали события и записывает их в лог

## Пример

```js
htmx.logger = function(elt, event, data) {
  if (console) {
    console.log("ИНФОРМАЦИЯ:", event, elt, data);
  }
}
```
