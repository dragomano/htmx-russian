Удаляет элемент из DOM.

## Параметры

* `elt` — элемент для удаления

или

* `elt` — элемент для удаления
* `delay` — задержка (в миллисекундах) перед удалением элемента

## Пример

```js
// удаляем my-div из DOM
htmx.remove(htmx.find("#my-div"));

// удаляем my-div из DOM после задержки в 2 секунды
htmx.remove(htmx.find("#my-div"), 2000);
```
