Переключает указанный класс у элемента.

## Параметры

* `elt` — элемент, у которого нужно переключить класс
* `class` — класс для переключения

## Пример

```js
// переключаем класс selected у элемента с id tab2
htmx.toggleClass(htmx.find("#tab2"), "selected");
```
