Вызывает заданное событие на элементе.

## Параметры

* `elt` — элемент, на котором нужно вызвать событие
* `name` — имя события, которое нужно вызвать
* `detail` — детали события

## Пример

```js
// вызываем событие myEvent на #tab2 с ответом 42
htmx.trigger("#tab2", "myEvent", {answer:42});
```
