Возвращает значения ввода, которые будут определены для данного элемента с помощью механизма разрешения значений htmx.

## Параметры

* `elt` — элемент, для которого нужно определить значения
* `request type` — тип запроса (например, `get` или `post`); не-GЕT запросы будут включать форму, содержащую элемент. По умолчанию используется `post`.

## Пример

```js
// получаем значения, связанные с этой формой
var values = htmx.values(htmx.find("#myForm"));
```
