Атрибут `hx-disable` отключает обработку htmx для указанного элемента и всех его дочерних элементов. Это может быть полезно как дополнительная мера безопасности при работе с HTML-контентом, когда:

1. На сайте используется контент, сгенерированный пользователями
2. Необходимо предотвратить атаки через вредоносные скрипты

Особенности работы:
- Значение атрибута игнорируется
- Действие не может быть отменено дочерними элементами

## Примечание

- `hx-disable` наследуется.
