Атрибут `hx-encoding` позволяет вам изменить кодировку запроса с обычной `application/x-www-form-urlencoded` на `multipart/form-data`, обычно для поддержки загрузки файлов в AJAX-запросе.

Значение этого атрибута должно быть `multipart/form-data`.

## Примечания

* `hx-encoding` наследуется и может быть размещён на родительском элементе.
