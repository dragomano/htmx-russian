Атрибут `hx-get` заставит элемент отправить `GET`-запрос по указанному URL и заменить HTML в DOM, используя стратегию замены:

```html wrap
<button hx-get="/example">Получить какой-то HTML</button>
```

Этот пример заставит `button` отправить `GET`-запрос на `/example` и заменить возвращаемый HTML в `innerHTML` элемента `button`.

## Примечания

- `hx-get` не наследуется.
- По умолчанию `hx-get` не включает никаких параметров. Вы можете использовать атрибут [hx-params](../hx-params), чтобы изменить это.
- Вы можете контролировать цель замены с помощью атрибута [hx-target](../hx-target).
- Вы можете контролировать стратегию замены, используя атрибут [hx-swap](../hx-swap).
- Вы можете контролировать, какое событие вызывает запрос, с помощью атрибута [hx-trigger](../hx-trigger).
- Вы можете контролировать данные, отправляемые с запросом, различными способами, описанными здесь: [Параметры](/ajax/parameters).
- Пустой `hx-get:""` сделает GET-запрос к текущему URL и заменит текущую HTML-страницу.
