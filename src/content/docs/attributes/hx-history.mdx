Установите атрибут `hx-history` в значение `false` для любого элемента в текущем документе или любого HTML-фрагмента, загружаемого в документ через htmx, чтобы предотвратить сохранение конфиденциальных данных в кеше localStorage при создании снимка состояния страницы.

Навигация по истории будет работать как обычно, но при восстановлении URL будет запрашиваться с сервера вместо использования кеша истории.

Пример:

```html
<html>
<body>
<div hx-history="false">
  ...
</div>
</body>
</html>
```

## Примечание

* `hx-history="false"` может находиться *где угодно* в документе, чтобы исключить текущее состояние страницы из кеша истории (даже вне элемента, указанного для снимка истории [hx-history-elt](/attributes/hx-history-elt)).
