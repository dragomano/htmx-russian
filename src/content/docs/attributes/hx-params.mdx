Атрибут `hx-params` позволяет вам фильтровать параметры, которые будут отправлены с AJAX-запросом.

Возможные значения этого атрибута:

* `*` — Включить все параметры (по умолчанию)
* `none` — Не включать никаких параметров
* `not <param-list>` — Включить все, кроме списка имён параметров, разделенных запятыми
* `<param-list>` — Включить только список имён параметров, разделённых запятыми

```html wrap
<div hx-get="/example" hx-params="*">
  Получить некоторый HTML, включая параметры
</div>
```

Этот div будет включать все параметры, которые были бы в `POST`, но они будут закодированы в URL и включены в URL, как обычно для `GET`.

## Примечание

* `hx-params` наследуется и может быть размещён на родительском элементе.
