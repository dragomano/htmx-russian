Атрибут `hx-post` заставит элемент отправить `POST`-запрос по указанному URL и заменить HTML в DOM, используя стратегию замены:

```html wrap
<button hx-post="/account/enable" hx-target="body">
  Включить учётку
</button>
```

Этот пример заставит `button` отправить `POST`-запрос на `/account/enable` и заменить возвращаемый HTML в `innerHTML` элемента `body`.

## Примечания

- `hx-post` не наследуется.
- Вы можете контролировать цель замены с помощью атрибута [hx-target](../hx-target).
- Вы можете контролировать стратегию замены, используя атрибут [hx-swap](../hx-swap).
- Вы можете контролировать, какое событие вызывает запрос, с помощью атрибута [hx-trigger](../hx-trigger).
- Вы можете контролировать данные, отправляемые с запросом, различными способами, описанными здесь: [Параметры](/parameters).
