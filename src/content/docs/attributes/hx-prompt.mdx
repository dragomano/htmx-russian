С помощью атрибута `hx-prompt` вы можете показать пользователю запрос на ввод перед выполнением действия. Значение, введенное в ответ на этот запрос, будет включено в заголовок `HX-Prompt`.

Вот пример:

```html
<button hx-delete="/account" hx-prompt="Введите имя аккаунта для подтверждения удаления">
  Удалить мою учётку
</button>
```

## Примечание

* `hx-prompt` наследуется и может быть размещён на родительском элементе.
