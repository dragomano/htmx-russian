Атрибут `hx-request` позволяет настроить различные аспекты запроса с помощью следующих параметров:

* `timeout` — таймаут запроса в миллисекундах
* `credentials` — будет ли запрос отправлять учётные данные
* `noHeaders` — удаляет все заголовки из запроса

Эти параметры задаются с использованием JSON-подобного синтаксиса:

```html
<div ... hx-request='{"timeout":100}'>
  ...
</div>
```

Для динамического вычисления значений можно добавить префикс `javascript:` или `js:`:

```html
<div ... hx-request='js: timeout:getTimeoutSetting() '>
  ...
</div>
```

## Примечание

* `hx-request` наследуется с объединением (merge-inherited) и может быть указан на родительском элементе.
