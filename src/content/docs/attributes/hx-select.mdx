Атрибут `hx-select` позволяет выбрать содержимое из ответа, которое вы хотите заменить. Значение этого атрибута — это CSS-селектор элемента или элементов, которые нужно выбрать из ответа.

Вот пример, который выбирает часть содержимого ответа:

```html
<div>
  <button hx-get="/info" hx-select="#info-detail" hx-swap="outerHTML">
    Получить информацию!
  </button>
</div>
```

Таким образом, эта кнопка выполнит запрос `GET` к `/info`, а затем выберет элемент с идентификатором `info-detail`, который заменит саму кнопку в DOM.

## Примечания

* Атрибут `hx-select` наследуется и может быть размещён на родительском элементе.
