Атрибут `hx-validate` заставляет элемент валидировать себя с помощью [HTML5 Validation API](https://htmx.org/docs/#validation) перед отправкой запроса.

По умолчанию только элементы `<form>` выполняют валидацию данных. Добавление `hx-validate="true"` к `<input>`, `<textarea>` или `<select>` включает валидацию перед отправкой запросов.

## Примечание

* `hx-validate` не наследуется.
